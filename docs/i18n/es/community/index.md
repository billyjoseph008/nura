# Comunidad

Participa en el ecosistema Nura.js, comparte ideas y colabora con el equipo principal.

- Lee la [guía de contribución](../../community/contributing.md) para entender el flujo de trabajo.
- Revisa el [código de conducta](../../community/code-of-conduct.md) y mantén un espacio respetuoso.
- Consulta la [política de seguridad](../../community/security.md) para reportar vulnerabilidades.
- Sigue la [guía de releases](../../community/releasing.md) si vas a publicar un paquete.
- Explora la [hoja de ruta](../../community/roadmap.md) y únete a las discusiones.
